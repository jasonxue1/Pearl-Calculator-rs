# Pearl-Calculator-rs

A Pearl Calculator for Minecraft

## Install

### GUI

- Download a prebuilt binary from [GitHub Releases](https://github.com/jasonxue1/Pearl-Calculator-rs/releases/latest)

- Build from source

  ```shell
  git clone https://github.com/jasonxue1/Pearl-Calculator-rs.git
  cd Pearl-Calculator-rs
  cargo build --rp pearl-calculator-gui
  ```

### CLI

The CLI package is published as `pearl-calculator` and installs the `pearl-calculator` binary.

- Install with `cargo`

  ```shell
  cargo install pearl-calculator
  ```

  Or using [`cargo-binstall`](https://github.com/cargo-bins/cargo-binstall)

  ```shell
  cargo binstall pearl-calculator
  ```

- Download a prebuilt binary from [GitHub Releases](https://github.com/jasonxue1/Pearl-Calculator-rs/releases/latest)

- Build from source

  ```shell
  git clone https://github.com/jasonxue1/Pearl-Calculator-rs.git
  cd Pearl-Calculator-rs
  cargo build -rp pearl-calculator
  ```
